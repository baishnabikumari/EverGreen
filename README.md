# Evergreen Decorator (decorate your christmas tree)

**Evergreen** is an interactive game where you can decorate a low-poly christmas tree in a snowy winter.

# videos


**Tools**
-- Three.js, js (core es6 module)
-- Css
-- HTML 5

## features
1. **3d environment**
2. **decoration mode with buttons**
3. **Snow fall**
4. **Delete mode**
5. **screenshot mode**

## How to play
1. Loading-- wait for 4sec
2. to take a view of 3d tree(rotate around tree use left click + drag OR Zoom in/ Out)
3. decorate the tree by using balls palette or use the gift button on bottom right - that gift box will open 3 button for STARS, BALLS and GIFT BOXES.
4. Trash- delete the entire decoration.
5. camera- take a shot of your decoration.

# how to RUN
1. clone repo and open the folder in vs code.
2. install the live server.
3. Right click on index.html and slect "open with Live server".
